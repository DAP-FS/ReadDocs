Simple Start: What is Computation?
----------------------------------

**Computation** is just following a set of steps to solve a problem.

.. raw:: html

   <div class="simple-example">
     <h4>ğŸ³ Real-Life Example: Making Toast</h4>
     <div class="example-steps">
       <div class="step-item">
         <span class="step-num">1</span>
         <span>Put bread in toaster</span>
       </div>
       <div class="step-item">
         <span class="step-num">2</span>
         <span>Set timer to 2 minutes</span>
       </div>
       <div class="step-item">
         <span class="step-num">3</span>
         <span>Wait for toast to pop up</span>
       </div>
       <div class="step-item">
         <span class="step-num">4</span>
         <span>Take out toast</span>
       </div>
     </div>
     <p class="example-note">
       <strong>That's computation!</strong> You followed clear steps to solve the "hungry" problem.
     </p>
   </div>

**In computer science terms:**
- **Input:** Bread + Hunger  
- **Process:** Toasting steps
- **Output:** Toast (problem solved!)
