Practice¶

This section provides structured practice problems for both weak and strong learners. Each problem includes multiple levels of hints and complete solutions.

Guided Practice for Weak Learners¶

Problem 1: Basic Construction¶

Task: Write a regular expression for the language of all strings over \(\{a, b\}\) that end with “ab”.

Hint 1 (Structure)

Think about this pattern: [anything][required ending] What should come before “ab”?

Hint 2 (Building Blocks)

  • For “anything over {a,b}”: use \((a + b)^*\)

  • For “must end with ab”: concatenate with \(ab\)

  • Combine: \((a + b)^*ab\)

Complete Solution

Answer: \((a + b)^*ab\)

Explanation: - \((a + b)^*\) generates any string over \(\{a, b\}\) (including \(\epsilon\)) - Concatenating with \(ab\) ensures every string ends with “ab” - Examples: \(\{ab, aab, bab, aaab, abab, baab, bbab, \ldots\}\)

Problem 2: Union Practice¶

Task: Create a regular expression for strings that either: - Contain only a’s, OR - Contain only b’s

Hint 1 (Breaking Down)

This is asking for a union of two languages: 1. Language 1: strings with only a’s 2. Language 2: strings with only b’s Use the + operator for union.

Hint 2 (Individual Parts)

  • Only a’s (including empty): \(a^*\)

  • Only b’s (including empty): \(b^*\)

  • Combine with union: \(a^* + b^*\)

Complete Solution

Answer: \(a^* + b^*\)

Language: \(L(a^* + b^*) = \{\epsilon, a, aa, aaa, \ldots\} \cup \{\epsilon, b, bb, bbb, \ldots\}\)

Note: \(\epsilon\) appears in both parts but union eliminates duplicates.

Problem 3: Star Operation¶

Task: Write a regular expression for strings over \(\{0, 1\}\) where every 0 is followed by at least one 1.

Hint 1 (Pattern Recognition)

What patterns are allowed? - Any number of 1’s alone: \(1^*\) - 0 followed by at least one 1: \(01^+\) (where \(1^+ = 11^*\))

How do we combine these?

Hint 2 (Building the Expression)

Think of valid strings as sequences of: - Standalone 1’s - Blocks of “0 followed by 1’s”

Pattern: \((1 + 01^+)^*\)

But we can simplify \(01^+\) as \(011^*\)

Complete Solution

Answer: \((1 + 011^*)^*\)

Alternative: \((1^* \cdot 011^*)^* \cdot 1^*\)

Explanation: - \(1\) allows standalone 1’s - \(011^*\) ensures every 0 has at least one following 1 - The star allows any number of these patterns - Examples: \(\{\epsilon, 1, 11, 01, 011, 0111, 101, 1011, \ldots\}\)

Intermediate Practice¶

Problem 4: Multiple Conditions¶

Task: Regular expression for binary strings that: - Have odd length, AND - Start with 1

Hint 1 (Length Analysis)

For odd length strings starting with 1: - Must start with 1 (1 symbol used) - Remaining length must be even - What generates even-length strings over \(\{0,1\}\)?

Hint 2 (Even Length Construction)

Even-length strings can be built as pairs: - \(((0+1)(0+1))^*\) generates all even-length strings - Alternatively: \((00 + 01 + 10 + 11)^*\)

So our pattern: \(1 \cdot \text{[even-length strings]}\)

Complete Solution

Answer: \(1((0+1)(0+1))^*\) or \(1(00+01+10+11)^*\)

Verification: - Length 1: \(\{1\}\) ✓ - Length 3: \(\{100, 101, 110, 111\}\) ✓ - Length 5: \(\{10000, 10001, 10010, \ldots, 11111\}\) ✓

Problem 5: Complement Thinking¶

Task: Strings over \(\{a, b\}\) that do NOT contain “aa” as a substring.

Hint 1 (State-Based Approach)

Think of states based on recent characters: - State 1: Just saw ‘b’ or at start (safe) - State 2: Just saw ‘a’ (danger - next ‘a’ is forbidden)

What transitions are allowed?

Hint 2 (Valid Patterns)

From state analysis: - Can always read ‘b’ - Can read ‘a’ only if not just after another ‘a’ - Valid patterns: start with b’s, then alternate a’s with b’s

Try: \(b^*(a b^*)^*\)

Complete Solution

Answer: \(b^*(ab^*)^*\)

Explanation: - \(b^*\): optional starting b’s - \((ab^*)^*\): any number of “a followed by b’s” - This ensures no two a’s are consecutive - Examples: \(\{\epsilon, a, b, ab, ba, bb, aba, abb, bab, bba, bbb, \ldots\}\)

Advanced Practice for Strong Learners¶

Problem 6: Modular Arithmetic¶

Task: Binary strings representing numbers divisible by 3.

Hint (State Machine Approach)

Use remainder states (mod 3): - State 0: remainder 0 (accepting - divisible by 3) - State 1: remainder 1 - State 2: remainder 2

Binary transitions: if current remainder is r and next bit is b, new remainder = (2r + b) mod 3

Solution Strategy

Build transition table:

State

On 0

On 1

0

0

1

1

2

0

2

1

2

Solve system of equations for state expressions.

Complete Solution

Answer: \((1(01^*0 + 10^*1)^*1)^* + (0 + 1(01^*0 + 10^*1)^*10^*)\)

Simplified: Through algebraic manipulation: \((0 + 1(01^*0 + 10^*1)^*)^*\)

This is complex - usually better to construct via NFA then convert!

Problem 7: Complex Constraints¶

Task: Strings over \(\{a, b, c\}\) where: - Number of a’s ≡ Number of b’s (mod 2) - Contains at least one c

Approach Hint

This requires tracking: 1. Parity of (count(a) - count(b)) 2. Whether we’ve seen a c

Four states needed: {even_diff_no_c, even_diff_has_c, odd_diff_no_c, odd_diff_has_c}

Solution Framework

Let E = even difference, O = odd difference, N = no c seen, Y = c seen

States: EN, EY, ON, OY (accepting: EY)

This leads to a complex regular expression - better solved algorithmically.

Challenge Problems¶

For the following problems, try to construct solutions independently:

Problem 8: Palindromes of even length over \(\{0, 1\}\)

Problem 9: Strings where every block of consecutive a’s has odd length

Problem 10: Binary strings where the number of 01 substrings equals the number of 10 substrings

Challenge Hints

Problem 8: Consider that palindromes can be built from center outward

Problem 9: Odd-length blocks look like \(a(aa)^*\)

Problem 10: This is equivalent to strings that start and end with same symbol!
